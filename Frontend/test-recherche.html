Guide interactif pour tester la recherche et le filtrage

Test 1 : Recherche Simple

Testez la recherche par nom de produit

Allez sur la page Catalogue
Tapez "basilic" dans la barre de recherche
Attendez 300ms ? Les résultats apparaissent automatiquement
Résultat Attendu : Tous les produits contenant "basilic" s'affichent

Test 2 : Filtrage par Catégorie

Testez le filtre par catégorie

Sélectionnez "Plantes Médicinales" dans le filtre
Observez le feedback visuel (bordure verte)
Résultat Attendu : Seules les plantes médicinales s'affichent avec badges verts

Test 3 : Tri des Produits

Testez les différentes options de tri

Sélectionnez "Prix croissant"
Vérifiez que les prix sont triés du plus petit au plus grand
Résultat Attendu : Les produits sont triés par prix croissant

Test 4 : Combinaison de Filtres

Testez la recherche + catégorie + tri

Recherche : "plante"
Catégorie : "Herbes Aromatiques"
Tri : "Nom A-Z"
Résultat Attendu : Herbes aromatiques contenant "plante", triées alphabétiquement

Test 5 : Raccourci Clavier

Testez la touche Échap pour effacer

Tapez quelque chose dans la recherche
Appuyez sur la touche "Échap"
Résultat Attendu : La recherche est effacée et tous les produits réapparaissent
