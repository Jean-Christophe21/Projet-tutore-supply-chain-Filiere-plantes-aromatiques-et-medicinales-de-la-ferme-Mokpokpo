? ?? COMMENT UTILISER                                                           ?
?????????????????????????????????????????????????????????????????????????????????
?                                                                               ?
?  Pour ADMIN:                                                                 ?
?  1. Se connecter sur admin-login.html                                        ?
?  2. Cliquer sur "Prédictions" dans le menu latéral                           ?
?  3. Voir les 2 sections :                                                    ?
?     • Prédictions des Ventes (en haut)                                       ?
?     • Données Historiques (en bas)                                           ?
?  4. Cliquer sur "Actualiser" pour recharger les données                      ?
?                                                                               ?
?  Pour GEST_COMMERCIAL:                                                       ?
?  1. Se connecter sur commercial-login.html                                   ?
?  2. Cliquer sur "Predictions Ventes" dans le menu latéral                    ?
?  3. Voir les prédictions de ventes générées par IA                           ?
?  4. Cliquer sur "Actualiser" pour recharger                                  ?
?                                                                               ?
?  Pour GEST_STOCK:                                                            ?
?  1. Se connecter sur stock-login.html                                        ?
?  2. Cliquer sur "Donnees Historiques" dans le menu latéral                   ?
?  3. Voir les données historiques pour analyse                                ?
?  4. Cliquer sur "Actualiser" pour recharger                                  ?
?                                                                               ?
