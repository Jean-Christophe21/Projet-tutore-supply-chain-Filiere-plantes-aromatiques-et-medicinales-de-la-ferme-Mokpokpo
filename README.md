# Projet-tutore-supply-chain-Filiere-plantes-aromatiques-et-medicinales-de-la-ferme-Mokpokpo
Ce dépôt contient tous les fichiers de code utilisé par le groupe 11 des étudiants en LF-GI pour leur projet tutoré.

## Structure du projet

Le projet est organisé en plusieurs dossiers :

- **`sql/`** - Contient le code SQL (scripts de création de base de données, requêtes, procédures stockées)
- **`web-app/`** - Contient l'application web (code source, configuration, dépendances)
- **`ui/`** - Contient les fichiers de l'interface utilisateur (HTML/CSS/JavaScript, assets, composants)
- **`autres/`** - Contient les autres fichiers (documentation, scripts utilitaires, ressources diverses)

Chaque dossier contient un fichier README.md avec plus de détails sur son contenu.
